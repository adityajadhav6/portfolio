## Overview

This project offers a scalable and secure healthcare platform, integrating EHRs, remote consultation, and machine learning analytics.

Demo link:https://tablet-alchemist-adityajadhav6-adityajadhav6s-projects.vercel.app

## Key Features
- **Search Medicines**: Search for medicines by entering a health condition (e.g., headache, cough, fever).
- **Medicine Recommendations**: Get a list of recommended medicines based on specific health conditions.
- **Nearby Clinich Finder**: Enter your location or a place name to discover nearby clinics using geolocation and map integration.
- **Tablet Details**: View detailed information about a specific tablet, including its name, image, and suitability for different age groups.
- **Autocomplete Suggestions**: Provides real-time autocomplete suggestions for health conditions and tablet names to enhance the user experience.
- **Interactive UI**: A clean and responsive user interface built with Bootstrap for seamless navigation.
- **Age Suitability Check**: Ensures that the recommended medicines are suitable for the user's age group.
- **Search Results Page**: Displays a list of tablets matching the search query with relevant details.
- **Expandable Database**: Uses a JSON-based database (drug_database.json) that can be easily updated with new medicines and conditions.
- **Unit Testing**: Includes unit tests to ensure the reliability and correctness of the core logic.
- **Future-Ready**: Designed with scalability in mind, allowing for easy integration of new features like user authentication, geolocation, and more.

## Technologies Used
- **Python: Core programming language for backend logic.
- **Flask: Lightweight web framework for building the application.
- **HTML5: For structuring the web pages.
- **CSS3: For styling and designing the user interface.
- **JSON: Used as a database to store medicine details.


The platform improves patient care through proactive monitoring and provides healthcare providers a unified view of records and predictions. It can be deployed on any major cloud platform.
